常见的异常类：
①IOExpetion：操作输入流和输出流时可能出现的异常
②ArithmeticException数学异常。如果把整数除以0，就会出现这种异常int a=12/0；
③NullPointerException空指针异常。当引用变量为null时，试图访问对象的属性或方法，就会出现这种异常，例如:
String d = null; system.out.println(d.length());
④ArraylndexOutOfBoundsException下标越界异常,—般出现在数组中，比如
int[]array = new int[4]; array[7]=1;
⑤ClassCastException,当类型转化失败的时候就会出现该异常
⑥lllegalArgumentException,非常参数异常，可以用来检查方法的参数是否合法
⑦NumberFormatException:在处理字符串和数字的转换中经常会遇到
